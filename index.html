안녕하세요👏
제 포트폴리오를 열람해 주셔서 감사합니다.

저는 실무에서 웹 퍼블리싱 업무만 중점으로 했던건 총 경력에서 4년 4개월 정도 해왔으며, 실무에서 html/css/jquery등을 사용해 마크업 작업 및 협업을 했습니다. 웹 퍼블리셔로 있었을 때에는 자바스크립트나 그외 접근성와 같은 것들을 다룰 기회가 크게 없었기 때문에 외부 스터디에 참여해 별도로 공부를 해왔고, 개인 포트폴리오에도 적용했습니다. 현재는 1차 목표인 프론트엔드 포지션에 도전하기 위해 공부중에 있습니다. 현재 가장 관심있고 중점적으로 연구하고 있는 것은 리액트 및 뷰, 타입스크립트 개발 그리고 테일윈드 CSS 입니다.

- 제가 가장 자신있게 할 수 있는 것은 시맨틱하고 유지 보수가 편리한 마크업과 CSS 레이아웃 구성입니다.

- 프론트엔드 개발자로서 필요한 것은 HTML,CSS,Vanilla Javascript실력을 바탕으로 프레임워크 활용과 UI interface구현하는 것이라 생각합니다. 또한 백엔드 요소(서버와 DB)가 프론트엔드 요소와 상호작용하는 프로세스를 이해하는 것도 중요하다고 생각합니다. 웹 스터디는 주로 책이나 블로그, 유료 인터넷 강의를 활용하며, 세미나 또는 오프라인 스터디에 참여하기도 했습니다.

- 평소에는 경제와 투자에 관심이 많아 경제 관련된 스터디나 모임을 주기적으로 하기도 했으며 지금은 무엇이 트렌드이고, 미래에는 어느 산업이 돌풍을 일으킬 것인지를 보는 것에 흥미를 가지고 있습니다.
